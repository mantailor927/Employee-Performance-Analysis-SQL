# Employee Performance Analysis Using SQL

## Project Overview

This project analyzes employee performance, salary distribution, attendance records, and department-wise performance using SQL.

The objective is to identify top-performing employees, high-paying departments, attendance trends, and employee rankings through business-focused SQL analysis.

---

## Tables Used

- Employees
- Departments
- Performance
- Attendance

---

- SELECT
- JOIN
- GROUP BY
- ORDER BY
- HAVING
- CASE
- SUM
- AVG
- COUNT
- MAX
- RANK()
- Window Functions
- Subqueries
- PARTITION BY

---

## Analysis Performed

### Department Salary Analysis

Analyzed:

- Employee Count
- Average Salary
- Total Salary Expense
- Maximum Salary

### High Salary Departments

Identified departments with average salary greater than 60,000.

### Salary Classification

Classified employees as:

- High Salary
- Normal Salary

### Employee Salary Ranking

Ranked employees based on salary using RANK().

### Highest Paid Employee Per Department

Identified the highest-paid employee in each department using PARTITION BY and RANK().

### Department Average Salary Comparison

Compared employee salaries against department average salary.

### Performance Classification

Classified employees into:

- Excellent
- Good
- Average

based on performance scores.

### Performance Ranking

Ranked employees according to performance score.

### Attendance Analysis

Classified employees into:

- Excellent Attendance
- Good Attendance
- Poor Attendance

### Best Performer In Each Department

Identified top-performing employee from every department.

### Department Performance Ranking

Ranked departments based on average performance score.

## Key Insights

- Sales department recorded the lowest average performance score.
- Harsh Patel achieved the highest performance score of 100.
